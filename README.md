# Beacon

## Link del proyecto publicado

https://beacon.com.mx/

## Descripcion

Sitio web para Beacon, desarrollado dentro del conjunto de proyectos de Servimex. Presenta informacion de marca, propuesta comercial y canales de contacto.

## Tecnologias usadas

- Astro
- React
- Tailwind CSS

## Requisitos

- Node.js 18+
- npm

## Retos tecnicos

- Construir una landing comercial con identidad visual propia.
- Organizar informacion de servicios y contacto en una estructura directa.
- Mantener componentes reutilizables para el ecosistema de sitios de Servimex.
- Preparar despliegue en dominio personalizado.
