# Beacon

## Link del proyecto publicado

https://beacon.com.mx/

## Resumen

Sitio web comercial para Beacon, una marca vinculada al ecosistema Servimex. El objetivo fue presentar la propuesta de valor, productos, contacto y agenda comercial en una experiencia clara, responsiva y facil de mantener.

## Rol

Desarrollado por Jonathan Alexis Bello Lopez.

Mi participacion se centro en la maquetacion del sitio, estructura de componentes, integracion de formularios y preparacion del despliegue en dominio personalizado.

## Tecnologias usadas

- Astro
- React
- Tailwind CSS
- Heroicons
- Formspree
- HubSpot Meetings

## Retos tecnicos

- Construir una landing comercial con identidad visual propia.
- Organizar productos, propuesta de valor y contacto en una estructura directa.
- Mantener componentes reutilizables para poder trabajar de forma consistente dentro del grupo de sitios Servimex.
- Integrar formularios de contacto con Formspree y agenda de presentaciones con HubSpot Meetings.
- Preparar el sitio para funcionar en dominio personalizado.
